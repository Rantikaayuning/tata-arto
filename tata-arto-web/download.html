Cara Install
Ikuti langkah-langkah di bawah
1
Download File APK
Tekan tombol "Mulai Download APK" di atas untuk mengunduh file
2
Aktifkan Instalasi Dari Sumber Tidak Dikenal
Buka Pengaturan → Aplikasi → Izin Khusus → Instal Aplikasi Tidak Dikenal, pilih file manager atau browser Anda, lalu aktifkan
3
Install APK
Buka file APK yang sudah diunduh, lalu tekan "Install"
4
Buat Akun atau Login
Buka aplikasi dan buat akun baru atau login dengan email Anda
